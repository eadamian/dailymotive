Frequently Asked Questions

How is this app personalized?

Daily Motive reaches to each user through their individual login accounts and a profile page. This provides access to their own videos and personalized profile page.

How do I create an account?

From the login screen, there is a button marked "Create an account". Fill out the text fields and your account will be made.

Will there be more features later on?

Daily Motive will continue to grow, entailing features like personal profile pictures, personalized choices on state of mood, and videos to motivate you more!
